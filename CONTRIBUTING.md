# 贡献指南

感谢你对 Aseprite 简体中文语言包的关注！

## 如何贡献

### 报告问题

如果你发现翻译错误或不准确的地方，请：

1. 在 [Issues](https://github.com/aseprite-chinese/aseprite-chinese/issues) 中创建新问题
2. 描述问题所在的位置（菜单路径、对话框名称等）
3. 说明当前翻译和建议的翻译
4. 如果可能，附上截图

### 改进翻译

1. Fork 本仓库
2. 编辑 `zh-hans.ini` 文件
3. 测试你的更改
4. 提交 Pull Request

### 翻译规范

- 保持术语一致性
- 使用简洁明了的中文
- 避免直译，注重意译
- 保持原文的语气和风格
- 技术术语可以保留英文或使用通用译法

### 常用术语对照

| 英文 | 中文 |
|------|------|
| Sprite | 精灵 |
| Layer | 图层 |
| Frame | 帧 |
| Cel | 单元格 |
| Palette | 调色板 |
| Canvas | 画布 |
| Brush | 画笔 |
| Tool | 工具 |
| Selection | 选区 |
| Mask | 遮罩 |

### 测试

在提交 PR 之前，请：

1. 在 Aseprite 中安装并测试你的翻译
2. 确保没有语法错误
3. 检查翻译在界面上的显示效果

## 行为准则

- 尊重他人
- 保持友善和专业
- 接受建设性的反馈
